Algorithms Used in FractureAI

1. Paris Law

Used to model fatigue crack growth based on stress intensity factor and crack length.

2. Stress Intensity Factor (SIF)

Calculates the threshold for crack propagation.

3. Cohesive Zone Models (CZM)

Models crack propagation using traction-separation laws.

4. AI-Based Models

Uses machine learning to predict crack growth and optimize simulation parameters.
